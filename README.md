# Language V1

Language V1 is a beginner-level custom programming language currently under development.

## Vision
The goal of Language V1 is to explore how programming languages work internally by building one from scratch.

## Current Features (2% Complete)
- Basic syntax structure (in progress)
- Initial parsing logic (in progress)

## What I'm Learning
- How compilers/interpreters work
- Syntax design and structure
- Logical execution of code

## Future Plans
- Add variables and data types
- Implement conditionals (if/else)
- Build a simple interpreter

## Why this project?
Instead of only learning programming, I wanted to understand how languages themselves are built.

This project is in early development and will be improved step by step.
